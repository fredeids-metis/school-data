# Fellesfag

Dette mappen inneholder markdown-filer for **obligatoriske fellesfag** (subjects mandatory for all students regardless of program).

## Struktur

Hver fellesfag har sin egen markdown-fil:

```
Norsk_VG2.md
Matematikk_1T.md
Historie_VG3.md
...
```

## Format

Samme format som programfag:

```markdown
---
id: norsk-vg2
title: Norsk VG2
fagkode: NOR1201
kategori: språk
lareplan: Læreplan i norsk
---

## Om faget
Beskrivelse av faget...

## Kompetansemål
- Mål 1
- Mål 2
```

## Forskjell fra Programfag

- **Programfag**: Valgfrie fag eleven velger selv
- **Fellesfag**: Obligatoriske fag påkrevd for alle elever
