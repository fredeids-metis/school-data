# Kommunikasjon og kultur 1

**Læreplan:** Læreplan i kommunikasjon og kultur

## Om faget

En reklame som fungerer i Norge, kan misforsta helt i et annet land. Hvorfor? I Kommunikasjon og kultur 1 utforsker du hvordan kulturelle faktorer pavirker kommunikasjon. Du analyserer tekster og medier, og du skaper egne kommunikasjonsuttrykk tilpasset ulike malgrupper. Faget gir deg kompetanse som er verdifull i alt fra journalistikk til markedsforing og kulturarbeid.

## Om faget - fra læreplan

Kommunikasjon og kultur fokuserer på hvordan elevene kan utvikle forståelse for kommunikasjon, språk og kulturelle perspektiver. Faget bidrar til at elevene utvikler bevissthet om hvordan kommunikasjon utformes og fortolkes i et kulturelt perspektiv. Kunnskapen ruster elevene til å bygge meningsfylte relasjoner, orientere seg i samfunnet og mestre framtidens arbeidsliv.

## Hvordan arbeider man i faget

I Kommunikasjon og kultur 1 arbeider du med å utforske, analysere og skape ulike former for kommunikasjon. Du undersøker hvordan kulturelle faktorer påvirker kommunikasjon, og hvordan kommunikasjon påvirker og formes av kultur. Undervisningen kombinerer teori med praktiske øvelser der du analyserer tekster, medier og kommunikasjonssituasjoner. Du lærer også å produsere egne tekster og kommunikasjonsuttrykk tilpasset ulike formål og målgrupper.

## Fagets relevans

Kommunikasjon og kultur passer for deg som er interessert i språk, kultur og hvordan mennesker kommuniserer. Faget gir kompetanse som er verdifull i mange sammenhenger - fra journalistikk og markedsføring til diplomati og kulturarbeid. Det er relevant for videre studier innen kommunikasjon, mediefag, kulturstudier, språkfag og samfunnsfag. Faget stimulerer kreativitet og utforskertrang gjennom møter med varierte tekstkulturer og medier.

## Kompetansemål

- utforske og analysere hvordan kulturelle faktorer påvirker kommunikasjon
- drøfte hvordan identitet og tilhørighet kommer til uttrykk gjennom språk og kommunikasjon
- analysere ulike typer tekster og medieuttrykk og vurdere hvordan de kommuniserer mening
- skape tekster og kommunikasjonsuttrykk tilpasset formål, mottaker og kanal
- utforske og drøfte sammenhenger mellom språk, makt og påvirkning
- anvende relevante begreper og teorier om kommunikasjon og kultur i analyse og drøfting
- reflektere over egen kommunikasjon og kulturelle bakgrunn
- samarbeide med andre og gi konstruktiv tilbakemelding på kommunikasjonsarbeid

## Kjerneelementer

### Kommunikasjon

Kjerneelementet kommunikasjon handler om hvordan mening skapes og utveksles mellom mennesker. Det handler om å forstå og bruke ulike kommunikasjonsformer og -kanaler, og om å tilpasse kommunikasjon til formål og mottaker.

### Kultur og identitet

Dette kjerneelementet handler om hvordan kultur former og uttrykkes gjennom kommunikasjon. Det handler også om hvordan identitet og tilhørighet kommer til uttrykk gjennom språk, symboler og andre kommunikasjonsformer.

### Tekst og medier

Kjerneelementet tekst og medier handler om å analysere og skape tekster i ulike sjangre og medier. Det handler om å forstå hvordan tekster og medier kommuniserer mening og påvirker holdninger og handlinger.

### Språk og makt

Dette kjerneelementet handler om sammenhenger mellom språk, makt og påvirkning. Det handler om å forstå hvordan språkbruk kan inkludere og ekskludere, og hvordan kommunikasjon kan brukes til å påvirke og overtale.
