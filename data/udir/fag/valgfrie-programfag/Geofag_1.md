# Geofag 1

**Læreplan:** Læreplan i geofag

## Om faget

Geofag handler om jordens oppbygning, prosesser og historie, samt samspillet mellom geosfærene og menneskets påvirkning på disse. Faget kombinerer naturvitenskap med forståelse av hvordan geologiske prosesser påvirker samfunnet og miljøet. Geofag gir innsikt i naturressurser, naturkatastrofer og klimaendringer.

## Hvordan arbeider man i faget

I Geofag 1 arbeider du med en kombinasjon av teori, feltarbeid og laboratorieøvelser. Du lærer å identifisere bergarter og mineraler, tolke geologiske kart og forstå landformer. Feltarbeid er en viktig del av faget der du undersøker geologiske fenomener i naturen. Digitale verktøy som GIS (geografiske informasjonssystemer) brukes til å analysere og visualisere geodata.

## Fagets relevans

Geofag passer for deg som er interessert i naturen, miljøet og hvordan jorden fungerer. Faget er relevant for videre studier innen geologi, geografi, miljøvitenskap, ressursforvaltning og petroleumsfag. Kunnskapen er også verdifull for å forstå samfunnsutfordringer knyttet til klimaendringer, naturkatastrofer og bærekraftig ressursbruk.

## Kompetansemål

- gjøre rede for jordas oppbygning og de viktigste bergartsdannende prosessene
- identifisere og klassifisere vanlige bergarter og mineraler og forklare hvordan de er dannet
- beskrive platetektonikk og forklare hvordan den former jordskorpen
- utforske og forklare hvordan ytre krefter former landskap gjennom forvitring, erosjon og avsetning
- bruke geologiske kart og profiler til å beskrive geologien i et område
- gjennomføre feltarbeid og laboratoriearbeid og dokumentere og presentere resultatene
- drøfte hvordan geologiske prosesser påvirker mennesker og samfunn
- utforske og vurdere hvordan naturressurser utnyttes og forvaltes

## Kjerneelementer

### Jorda som system

Kjerneelementet jorda som system handler om hvordan de ulike geosfærene (litosfæren, hydrosfæren, atmosfæren og biosfæren) samvirker og påvirker hverandre. Det handler om å forstå jorden som et dynamisk system der energi og materie utveksles mellom sfærene.

### Geofaglige metoder og verktøy

Dette kjerneelementet handler om å bruke geofaglige metoder og verktøy for å undersøke og forstå geologiske fenomener. Det inkluderer feltarbeid, laboratoriearbeid, bruk av kart og digitale verktøy, samt vitenskapelig metode.

### Naturressurser og bærekraft

Kjerneelementet naturressurser og bærekraft handler om hvordan geologiske ressurser dannes, utnyttes og forvaltes. Det handler også om å vurdere miljøkonsekvenser og bærekraftig ressursbruk.

### Naturfare og samfunn

Dette kjerneelementet handler om hvordan geologiske prosesser kan utgjøre fare for mennesker og samfunn, og hvordan samfunnet kan tilpasse seg og forebygge skader fra naturhendelser.
